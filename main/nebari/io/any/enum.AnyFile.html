pub enum AnyFile {
    Std(StdFile),
    Memory(MemoryFile),
}
Expand description

A file that can be either a StdFile or MemoryFile.

Variants

Std(StdFile)

A file backed by a filesystem.

Memory(MemoryFile)

A simulated file backed by memory.
